# Поиск клиента в Mindbox

Когда пользователь просит найти клиента в Mindbox:

1. Определи идентификатор: email, телефон или external_id
2. Вызови инструмент `get_customer` с нужным параметром
3. Если клиент найден — покажи профиль и сегменты
4. Если не найден — предложи другой идентификатор

## Примеры запросов
- "Найди клиента с email user@example.com"
- "Покажи профиль клиента +79001234567"
- "Кто клиент с ID 12345?"

## Формат ответа
Покажи данные клиента в читаемом виде:
- ФИО, email, телефон
- Дата рождения, пол
- Регион
- Список сегментов (если есть)
