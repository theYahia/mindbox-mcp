# Статистика сегментов Mindbox

Когда пользователь просит показать сегменты клиента:

1. Определи идентификатор клиента: email, телефон или external_id
2. Вызови инструмент `get_segments`
3. Покажи список сегментов с количеством клиентов в каждом

## Примеры запросов
- "Статистика сегментов для user@example.com"
- "В каких сегментах клиент +79001234567?"
- "Покажи сегменты клиента"

## Формат ответа
Таблица сегментов:
- Название сегментации
- Название сегмента
- Количество клиентов (если доступно)
